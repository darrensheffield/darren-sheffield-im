Private workspace pointer

Darren private area entry point

The private area is intentionally local-only. GitHub Pages can not reach local files or iCloud Drive directly, so this page only points to a companion server that you run yourself.

Open local private area

Start the local private server for the companion app. The button below works on the same Mac.

Open local dashboard 
cd /Users/darrensheffield/Documents/Automation/darren-sheffield-im/darren-private-local
python3 -m http.server 4173

For iPad or iPhone testing, open the Mac's local network address instead, such as http://your-mac.local:4173/. Use CloudKit or HTTPS before real private data is exposed.

Scope note

The local scaffold below is designed for development only. It uses sample JSON and a local lock screen interface. No real auth, private tokens, or CloudKit credentials are currently active.

Do not treat this as a secure production path until CloudKit or an authenticated local endpoint is wired.
